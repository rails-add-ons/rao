## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rao-resource_controller'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rao-resource_controller
```

Generate the initializer:

```bash
$ rails g rao:resource_controller:install
```
